# BCube Technologies Website

BCube Technologies is a product studio that builds single-purpose applications, each designed to solve one problem exceptionally well. This repository contains the company's marketing website and brand assets.

The website is a modern single-page site built with Astro 5 and Tailwind CSS 4, showcasing BCube's philosophy of focused, bloat-free software. It features a full-viewport hero section, company mission ("We hate bloated apps too"), a product showcase for Lawh — an AI-powered Quran verse recognition app that provides instant Arabic text, transliteration, and translation from any recitation — along with an early access signup, FAQ accordion, and contact form. The design follows BCube's brand guide: Dark Blue (#0B0F8A), Pastel Azure (#7DB9D3), and clean white/gray backgrounds with Poppins typography.

The repository also includes brand assets (SVG logos in wordmark and pictorial variations across multiple colorways), a color design guide, and marketing copy documentation.

The site is fully static, responsive across mobile/tablet/desktop, and uses vanilla JavaScript for interactive elements like the mobile navigation menu and FAQ accordion. Forms are currently client-side only, ready to be connected to a backend email service. Built for performance and simplicity — no unnecessary dependencies, no JavaScript frameworks, just clean static HTML.

## Getting Started

```bash
npm install
npm run dev
```

The site runs at `http://localhost:4321/`.

## Build

```bash
npm run build
npm run preview
```
